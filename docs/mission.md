# MISSION.md

## 🎯 Mission

Build a high-quality, scalable content generation system for an Indian epics quiz app.

This tool is the **content engine** — not the user-facing product.

---

## 💡 Problem

- Manual question creation does not scale
- AI-only generation is unreliable
- Need high-quality, verified, source-grounded content

---

## ✅ Solution

A hybrid system:
- AI generates questions (80%)
- Human reviews and approves (20%)

---

## 🧩 Core Workflow

Input → AI Generate → [Grounding Validation] → [Self-Critique] → Draft Store → Human Review → Approved Store

---

## 🎯 Success Criteria

- 300+ approved questions
- Each question is:
  - Source grounded (validated by second LLM pass)
  - Engagement-scored (self-critique pass, 1-10)
  - Story-phase tagged (Early Life → Return and Reunion)
  - Reviewed and approved by human

---

## 🚀 Long-Term Vision

- Expand to 1000+ questions
- Add Hindi support
- Add user-facing quiz app
- Introduce personalization later

---

## ⚠️ Constraints

- Solo developer
- Limited time
- Must ship fast

---

## 🧠 Strategic Principle

Content quality is the moat.

Infrastructure is secondary.
